# 🕒 DarkTime | The High-Contrast Digital Lab

A hyper-minimalist, industrial-grade digital clock engineered for focus. **DarkTime** strips away the noise, delivering a high-contrast, deep-black interface designed for developers, creators, and technical workstations.

---

## ✨ What’s New in v5.0
* **Double-Tap Immersion:** Double-click anywhere to trigger true fullscreen mode for a distraction-free environment.
* **Industrial Typography:** Integrated **JetBrains Mono** for a high-precision, technical monospace aesthetic that ensures perfect character alignment.
* **Dynamic Feedback:** Integrated glassmorphism toast notifications for interaction feedback, featuring backdrop blur and subtle animations.
* **Zero-Distraction UI:** Responsive "Hidden Dot" logic for mobile portrait views to maintain a clean vertical stack and maximize readability on small screens.

---

## 🚀 Core Features
* **Precision Tracking:** Real-time 1:1 synchronization with local system hardware.
* **OLED Optimized:** Deep black architecture (#000000) to reduce power consumption and eye strain.
* **Fluid Responsiveness:** Adaptive scaling logic using CSS `clamp()` and `vw` units from 4K monitors to mobile devices.
* **Offline First:** No external dependencies; runs entirely on-device using local JavaScript.
